Electronics

We have used a ESP32 to control evrything. we used Ultrasonic sensor to sense the objects infront which is going to help us to get the data that we need to control the game. we used simple LED setup to indicate the connnections like WIFI connection and server Conncetion. We kept Hardware simple and neat to avoid any error which might take Days to fix.

Communication

As communication is vital in our project and we need a very fast data transfer speed, we had to use websockets to send and recive data.
We used Django first which was slow,so we have to switch to websockets. Hosted a websocket server in ESP32 and using Localnetworks and using its IP we were able to communicate with the Game and control it.
To know how games communicate => Here

How Games Made

Game are made using HTML,CSS and JavaScript. There is a central JS hub which is responsible for communicating with all the games. All the data that goes into games are coming from this central hub. Then games take that data and act accordingly.
